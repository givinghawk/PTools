# PTools

Update and enable/disable maintainence mode faster than ever with PTools!
<br>
Usage of PTools

```bash

curl -Ls https://raw.githubusercontent.com/givinghawk/PTools/main/ptools.sh | grep -wo "https.*ptools.sh" | wget -qi -
```

```bash

bash ptools.sh
```
